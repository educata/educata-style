# Spinner

The Spinner component is a loading indicator that provides visual feedback to users while content is being loaded or processed.
It can be used in various scenarios, such as during data fetching, form submission, or any other asynchronous operation.

## Variations

There are several variations of the Spinner component that can be used to customize its appearance and behavior.

There are total of 7 color variation.

The Spinner component can be resized using the `size` prop. The available sizes are `sm`, and `lg`.
